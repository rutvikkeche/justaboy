# Project Overview

This project was created to formally ask a very special girl to be my partner, inspired by our shared love for *Notting Hill*, whimsical art styles, and unapologetically cheesy romantic proposals.
What you see is a camera playing a grainy scene from one of Hugh Grant’s films, specifically *Four Weddings and a Funeral*, featuring the moment where he confesses his love to Carrie. It perfectly captures the kind of romance this project was built around.
